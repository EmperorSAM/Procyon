Контент по середине страницы

Чтобы материал на странице был по середине экрана, поместите его в <div class="row"> ... </div>. Этот элемент будет менять ширину в зависимости от типа экрана. Как на этом сайте.

Если Вы хотите использовать "резиновую" верстку, то просто не используйте данный класс. Но если вдруг понадобилось переключаться на "резиновую" верстку для некоторых дисплеев, то существую следующие классы. Их необходимо добавить в элемент, пример <div class="row fluid-md-up"> ... </div>.

  • .fluid-sm - "резиновый" row только для планшетов.
  • .fluid-sm-up - для планшетов и больше.
  • .fluid-sm-down - для планшетов и меньше.
  • .fluid-md - только для средних дисплеев.
  • .fluid-md-up - для средних дисплеев и больше.
  • .fluid-md-down - для средних дисплеев и меньше.
  • .fluid-lg - только для больших дисплеев.
  • .fluid-landscape - для альбомной ориентации
  • .fluid-portrait - для портретной ориентации
  • .fluid-4by3 - для диагонали дисплеев 4/3
  • .fluid-16by9 - для диагонали дисплеев 16/9
  • .fluid-16by10 - для диагонали дисплеев 16/10
  • .fluid-16by9and10 - для диагонали дисплеев 16/9 и 16/10
  • .fluid-print - для режима печати

Класс .row всегда имеет ширину 100% для смарфонов (.xs-#).

Вложенные .row

  • Певый .row в разметке - по середине страницы (см. пункт).
  • .row > .row (прямой потомок) - у второго .row отступа не будет. Можно добавлять другие блоки.
  • [xs-# sm-# md-# ld-# block-*] > .row (прямое наследование) - у второго .row появляются отпицательные левые и правые отступы для выравнивания отступов. Поэтому его надо помещать непосредственно в адаптивные ячейки .xs-#. (см. пункт)

  • sm-center-down - Центрирование для планшетов и ниже.

Режимы показа. Где * вышеперечисленные классы.

  • * (без дополнений, по умолчанию) - {display: inherit !important;}
  • *-block {display: block !important;}
  • *-inline {display: inline !important;}
  • *-inline-block {display: inline-block !important;}

Пример:

  • .show-inline - Показать в линию для всех дисплеев.
  • .show-print-inline - Показать в линию в режиме печати.
  • .show-4by3-inline - Показать в линию для мониторов диагональю 4/3.
  • .sm-show-up-block - Показать для планшетов и выше в режиме block.

Дополнительные классы

Чтобы воспользоваться дополнительными классами, необходимо просто их добавить к элеменнту. Значения по умолчанию можно легко сменить в less исходниках.

Класс CSS Описание
.radius border-radius: 5px; Закругление границ любого элемента. По умолчанию 5px.
.round border-radius: 1000px; Закругление границ любого элемента. По умолчанию 1000px.
.circle border-radius: 50%; Закругление границ любого элемента до круга.
.left float: left !important; Придать элементу свойство float: left.
.right float: right !important; Придать элементу свойство float: right.
.clearfix clearfix: both; Применить для элемента обтекание со всех сторон.
.no-padding padding: 0 !important; Убрать у элемента все внутрение отступы.

Модальные окна

Открыть модальное окно

Чтобы открыть модальное окно, необходимо к элементу добавить data-modal-id="myModal", где myModal - это id открываемого модального окна.

Размеры окна

Класс размера окна добавляется к основному контейнеру модального окна. Пример <div id="myModal" class="modal-xs" data-modal>.

Если не задавать определенный размер, то ширина составит 80%.

Все модальные окна на маленьких экранах имеют 100% ширины и высоты.

  • .modal-xs - ширина 30%
  • .modal-sm - ширина 40%
  • .modal-md - ширина 60%
  • .modal-lg - ширина 90%
  • .modal-full - на весь экран

JS

С использованием jquery, можно взаимодействовать следующим образом:

                
                    // Через событие
                    $('a.modal-link').triger('click');
                    $('a.modal-link-close').triger('click');
                    
                    // Вызвать или закрыть модальное окно
                    $('#myModal').modal();
